Иерархия логов сервера. Родительские логи содержат сообщения потомков. Корневой лог отображает все сообщения, которые генерирует сервер.

"server"
	"server-db" -- все запросы к БД
	"server-server" -- работа с сетевыми соединениями, создание клиентских потоков
		"server-client" -- обработка сообщений клиентов (бизнес логика)
			"server-client-<UNP>" -- обработка сообщений для одного клиента
	"server-teller" -- поток обработки транзакций
